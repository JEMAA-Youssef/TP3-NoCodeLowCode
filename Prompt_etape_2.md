# Prompt Étape 2 --- Génération dynamique des anneaux (Tours de Hanoï)

Ajoute l'Étape 2 à mon projet **Tours de Hanoï**.

Voici ce que je veux que tu ajoutes à mes fichiers existants
(HTML/CSS/JS de l'étape 1) :

Quand je clique sur le bouton « **Démarrer** », tu dois :

-   récupérer le **nombre d'anneaux** dans l'input,\
-   **vider la première tour**,\
-   **générer automatiquement les anneaux** dans la première tour,\
-   les **empiler correctement** (grand en bas → petit en haut),\
-   donner à chaque anneau une **largeur différente**,\
-   utiliser la fonction **startGame()** pour lancer la génération.

### Important :

-   **Ne modifie pas** ce qui existe déjà dans l'étape 1.\
-   Ajoute **seulement** ce qui est nécessaire.\
-   Donne-moi **uniquement** le code à ajouter ou remplacer dans mes
    fichiers existants.
