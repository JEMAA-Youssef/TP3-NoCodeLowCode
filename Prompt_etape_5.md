# Prompt Étape 5 --- Démo Automatique (Tours de Hanoï)

Ajoute l'Étape 5 de mon projet Tours de Hanoï.

##  Objectif

Ajouter un bouton **« Démo automatique »** qui :

-   lance la résolution automatique du jeu,
-   de manière **visible**, avec un délai entre chaque mouvement,
-   en utilisant **l'algorithme récursif classique** des Tours de Hanoï.

##  Travail demandé

Donne-moi uniquement :

-   le **HTML** pour ajouter le bouton « Démo automatique »,
-   le **JavaScript** nécessaire pour :
    -   implémenter l'algorithme récursif `hanoi(n, from, to, aux)`,
    -   enregistrer les mouvements,
    -   exécuter un mouvement à la fois avec un délai (ex : 300--600
        ms),
    -   déplacer réellement les anneaux en utilisant ma logique de
        l'étape 3,
-   les **modifications CSS légères** si nécessaires.

##  Restrictions

-   Ne réécris pas les étapes précédentes.
-   N'altère pas le déplacement manuel.
-   La résolution **ne doit pas être instantanée**, mais visible étape
    par étape.
